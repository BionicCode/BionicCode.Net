TextControl Attached Properties

The TextControl type exposes the following members.

Attached Properties
  NameDescription
HighlightBackground
Attached property to define the background Brush for the highlight text, which is defined by HighlightRange items contained in the HighlightRangesProperty attached property.
HighlightForeground
Attached property to define the foreground Brush for the highlight text, which is defined by HighlightRange items contained in the HighlightRangesProperty attached property.
HighlightRanges
Attached property to define a HighlightRangeCollection of HighlightRange items. cref="HighlightRangesProperty"/> attached property.
IsHighlightingEnabled
Attached property to enable or disable the highlight attached behavior TextControl. cref="HighlightRangesProperty"/> attached property.
Text
Attached property to serve as alternative text property for the RichTextBox (instead of using Document). Optional property to use with TextBlock (instead of Text). The defined HighlightRange items contained in the attached property HighlightRangesProperty collection will always be applied to Text and the TextProperty values.
