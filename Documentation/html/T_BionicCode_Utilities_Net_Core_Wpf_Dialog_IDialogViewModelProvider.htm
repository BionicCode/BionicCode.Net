Remarks
The IDialogViewModel can be received from instances that implement IDialogViewModelProviderSource by subscribing to their DialogRequested event.
            When used together with the attached DialogDataContextProperty the setting of the DialogViewModel property automatically triggers the showing of a dialog. The IDialogViewModel is rendered by a custom defined [!:DataTemplate] that targets the concrete type of a IDialogViewModel implementation.
