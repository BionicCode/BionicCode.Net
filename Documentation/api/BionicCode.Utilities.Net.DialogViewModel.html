    ResponseCallbackAsync

    The asynchronous callback that is invoked during the dialog interaction. Use this callback as continuation of the interrupted flow, after the required data was collected by the dialog.

    Declaration
    public Func<IDialogViewModel, Task> ResponseCallbackAsync { get; set; }
    Property Value
    Type Description
    System.Func<IDialogViewModel, System.Threading.Tasks.Task>
    Remarks

    When using the abstract class DialogViewModel this delegate is invoked when invoking the SendResponseAsyncCommand. The parameter of this callback is the original IDialogViewModel. The callback can access the DialogResult property to get the interaction result. The ResponseCallbackAsync makes the dialog interaction fire-and-forget as the IDialogViewModelProviderSource doesn't need to wait for the dialog to close and doesn't have to store a reference to the IDialogViewModel.

    Remarks

    When using the abstract class DialogViewModel the SendResponseAsyncCommand sets the DialogResult property and invokes the ResponseCallbackAsync continuation callback. The parameter of this callback is the original IDialogViewModel. The callback can access the DialogResult property to get the interaction result. The ResponseCallbackAsync makes the dialog interaction fire-and-forget as the IDialogViewModelProviderSource doesn't need to wait for the dialog to close and doesn't have to store a reference to the IDialogViewModel.

    OnInteractionCompleting(CancelEventArgs)

    Event invocator of the InteractionCompleting event. Raised before InteractionCompleted event. When overriding the member, setting the property System.ComponentModel.CancelEventArgs.Cancel of the cancelEventArgs parameter to true will cancel the dialog close process.

    Declaration
    protected virtual void OnInteractionCompleting(CancelEventArgs cancelEventArgs)
    Parameters
    Type Name Description
    System.ComponentModel.CancelEventArgs cancelEventArgs

    When setting the property System.ComponentModel.CancelEventArgs.Cancel of the event's System.ComponentModel.CancelEventArgs to true will cancel the dialog close process.
    Remarks

    When the dialog close process was cancelled e.g., by overriding the OnInteractionCompleting(CancelEventArgs) method and by setting the cancelEventArgs to true, a new close process can be triggered either by invoking the OnInteractionCompleted() method or by invoking the SendResponseAsyncCommand.
