    LastInSorted<TItem>(IEnumerable<TItem>, Func<TItem, Boolean>)

    A non-cached version of Last<TSource>(IEnumerable<TSource>, Func<TSource, Boolean>) for ascending sorted collections.

    Declaration
    public static TItem LastInSorted<TItem>(this IEnumerable<TItem> source, Func<TItem, bool> predicate)
    Parameters
    Type Name Description
    IEnumerable<TItem> source
    Func<TItem, Boolean> predicate

    A delegate to test each element for a condition.
    Returns
    Type Description
    TItem

    The last element in a sorted collection that satisfies the predicate delegate. If no such an element was found, a InvalidOperationException exception will be thrown.
    Type Parameters
    Name Description
    TItem
    Remarks

    The collection is expected to be sorted. Otherwise this method can yield unexpected results. The search will stop after the last consecutive match.
    While the standard Last<TSource>(IEnumerable<TSource>, Func<TSource, Boolean>) iterates and caches the complete collection in order to produce a correct result for unsorted collections, LastInSorted<TItem>(IEnumerable<TItem>, Func<TItem, Boolean>) expects a sorted collection to avoid iterating the complete collection and therefore to significantly improve the performance in terms of speed and memory footprint.

    The performance gain is only relevant if the collection is sorted in ascending order.

    Exceptions
    Type Condition
    ArgumentNullException

    source is null.
    ArgumentNullException

    predicate is null.
    InvalidOperationException

    The source collection does not contain any element that satisfies the predicate delegate.

    The source collection is empty.

    Parameters
    Type Name Description
    IEnumerable<TData> source

    The source collection to convert to a DataTable.
    Returns
    Type Description
    DataTable

    A DataTable created from public properties of the objects contained in the source.
    Type Parameters
    Name Description
    TData

    The type of the objects stored in the source collection.
    Remarks

    The extension method creates a DataTable from the source object's public properties.
    Each public property is translated to a column where the property name is the column name.
    To control column naming, you can decorate the property with the DisplayNameAttribute attribute to provide an alternative column name for the decorated property.

    To exclude public properties from the conversion, decorate them with the IgnoreAttribute attribute.
