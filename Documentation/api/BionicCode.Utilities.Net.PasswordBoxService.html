    Class PasswordBoxService

    Attached behavior for System.Windows.Controls.PasswordBox that will send the System.Security.SecureString of the System.Windows.Controls.PasswordBox.SecurePassword property to a command target e.g., view model using a registered System.Windows.Input.ICommand registered with the CommandProperty attached property.

    Remarks

    The attached behavior does at no point unwrap the the System.Security.SecureString returned from the System.Windows.Controls.PasswordBox.SecurePassword property, nor does it access the security critical System.Windows.Controls.PasswordBox.Password property. The System.Windows.Controls.PasswordBox.SecurePassword value is simply forwarded to the command target of the registered CommandProperty attached property.
