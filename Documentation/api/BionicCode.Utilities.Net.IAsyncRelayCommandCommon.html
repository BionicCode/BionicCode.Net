    Interface IAsyncRelayCommandCommon

    An interface to define a reusable command that encapsulates the implementation of ICommand with support for async/await command delegates and data binding (implements INotifyPropertyChanged).
    Enables instant creation of an ICommand without implementing the ICommand interface for each command.
