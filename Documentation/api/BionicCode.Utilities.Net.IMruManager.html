    MaxMostRecentlyUsedCount

    The maximum number of files that are kept in the MRU table.
    The default value is 10.

    Declaration
    int MaxMostRecentlyUsedCount { get; set; }
    Property Value
    Type Description
    System.Int32
    Remarks

    When the limit is exceeded, the least recent used file will be removed from the MRU table every time a new file is added.
    The maximum allowed value is 100. The minimum allowed value is 1.

    FileAdded

    Raised when a new file was added to the MRU list. The event args contains the old and the new MostRecentlyUsedFileItem.
    Once the max MostRecentlyUsedFileItem limit is reached, the least used file will be removed from the list to make space for the new item.
    In this case the BionicCode.Utilities.Net.ValueChangedEventArgs`1.OldValue is the removed oldest item in the list.
    Otherwise BionicCode.Utilities.Net.ValueChangedEventArgs`1.OldValue will be null.
