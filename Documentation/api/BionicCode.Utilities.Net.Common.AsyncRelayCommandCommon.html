    Class AsyncRelayCommandCommon

    A reusable command that encapsulates the implementation of ICommand with support for async/await command delegates.
    Enables instant creation of an ICommand without implementing the ICommand interface for each command. The AsyncRelayCommandCommon<TParam> accepts asynchronous command handlers and supports data bindng to properties like IsExecuting by implementing INotifyPropertyChanged.
    Call and await the ExecuteAsync() method or one of its overloads to execute the command explicitly asynchronously. ICommand

    Syntax
    public abstract class AsyncRelayCommandCommon : IAsyncRelayCommandCommon, ICommand, INotifyPropertyChanged
    Remarks

    AsyncRelayCommandCommon implements ICommand. In case the AsyncRelayCommandCommon<TParam> is executed explicitly, especially with an asynchronous command handler registered, it is highly recommended to invoke the awaitable ExecuteAsync() or its overloads instead.
