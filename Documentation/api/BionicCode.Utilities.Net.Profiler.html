    LogTimeAsync(Func<Task>, Int32)

    Measures the execution time of a method.

    Declaration
    public static Task<ProfilerBatchResult> LogTimeAsync(Func<Task> asyncAction, int runCount)
    Parameters
    Type Name Description
    Func<Task> asyncAction

    A delegate that executes the asynchronous code to measure execution time.
    Int32 runCount

    Number of iterations the asyncAction should be executed.
    Returns
    Type Description
    Task<ProfilerBatchResult>

    A Task holding the average execution time of all runCount number of iterations as TimeSpan.
    Remarks

    Cancelled tasks are ignored when calculating the result (although the cancelled runs are listed in the Summary, but marked as cancelled).
    A cancelled task is a Task where the Status returns either Canceled or Faulted or an OperationCanceledException exception was thrown.

    Use the LogTimeAsync(Func<Task>, Int32, ProfilerLogger) overload and provide an instance of the ProfilerLogger delegate for the logger parameter to control the output target or customize the formatting.
