    Enum AccessModifier

    Namespace: BionicCode.Utilities.Net
    Assembly: BionicCode.Utilities.Net.Common.dll
    Syntax
    public enum AccessModifier
    Remarks

    The enum values are ordered from least restrictive to most restrictive. Therefore, the hierarchy can be determined by comparing their integral numeric.
    For example, using the less than operator < to compare two AccessModifier values like Console.WriteLine(AccessModifier.Public < AccessModifier.Private) would print "True", expressing that langword_csharp_Public is less restrictive than langword_csharp_private.
