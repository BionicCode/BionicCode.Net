    AddColumn<TData>(DataTable, String, Int32)

    Adds a new DataColumn to a DataTable.

    Declaration
    public static void AddColumn<TData>(this DataTable source, string columnName, int columnIndex)
    Parameters
    Type Name Description
    DataTable source

    The DataTable to add the new DataColumn to.
    String columnName

    The column's name.
    Int32 columnIndex

    The explicit index of the created DataColumn.
    Type Parameters
    Name Description
    TData

    The data type of the column value.
    Remarks

    If columnIndex is less than 0 or greater than the existing number of columns - 1 (greater than the ordinal of the last column) then an Invalid ArgumentException is thrown.

    Any columns between the previous and new ordinal will be renumbered, to adjust for a column's new ordinal.

    Added columns are initialized to the default value of TData for all existing rows.
    AddColumn<TData>(DataTable, String, Int32) should be called before an overload of AddRow(DataTable, Object[])
