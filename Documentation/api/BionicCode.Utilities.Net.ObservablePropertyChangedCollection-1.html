    Class ObservablePropertyChangedCollection<TItem>

    Raises CollectionChanged event when the property of an item raised PropertyChanged. The NotifyCollectionChangedAction for this particular notification is Reset with a reference to the notifying item and the item's index. .

    Remarks

    The item must implement INotifyPropertyChanged otherwise the behavior is like a common ObservableCollection<T>. The ObservablePropertyChangedCollection<TItem> implements the weak event pattern in order to handle the PropertyChanged event.
