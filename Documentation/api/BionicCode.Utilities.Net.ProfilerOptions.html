    BaseUnit

    The default base unit that all results are transformed to.

    Declaration
    public TimeUnit BaseUnit { get; set; }
    Property Value
    Type Description
    TimeUnit

    The default is defined by DefaultBaseUnit.
    Remarks

    Internally time is recorded in microseconds and then converted to the desired base unit (e.g., ms). The usual timer resolution on most modern machines is 100 ns (see Ticks to learn more. Call the static IsHighResolution field to know if the executing machine supports that high resolution.)
