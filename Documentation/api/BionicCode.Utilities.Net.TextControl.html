    TextProperty

    Attached property to serve as alternative text property for the RichTextBox (instead of using Document). Optional property to use with TextBlock (instead of Text). The defined HighlightRange items contained in the attached property HighlightRangesProperty collection will always be applied to Text and the TextProperty values.

    Declaration
    public static readonly DependencyProperty TextProperty
    Field Value
    Type Description
    DependencyProperty
    Remarks

    In case of the TextProperty being attached to a RichTextBox, the string value will be converted to a FlowDocument and assigned to the Document property.
