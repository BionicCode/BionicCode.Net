    DisableWarmup()

    Disable warm up iterations. Warm up iterations can be scheduled to trigger the JIT compiler. Warm up is enabled by default.

    Declaration
    public ProfilerBuilder DisableWarmup()
    Returns
    Type Description
    ProfilerBuilder

    The currently configured ProfilerBuilder instance to enable to chain calls.
    Remarks

    When running code the first time there is always the incurrence of the JIT to compile the code.
    For this reason it is recommended to execute the code at least once in order to avoid the JIT to impact the profiling.
