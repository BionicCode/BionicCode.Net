    Remarks

    This attribute can be used in case the type has no default constructor, or profiling requires a specially initialized instance.
    The default constructor must be an instance constructor and can be defined with any access modifier (e.g. langword_csharp_public or langword_csharp_private).
    The factory member must be static and can be a method, property or field and must return an instance of the type that declares the profiled memeber.


    The Profiler creates an instance of the profiled type using one of the following providers ranked by prcedence:

    1. It first checks if the type has a static member that is decorated with the ProfilerFactoryAttribute and returns the same type as the type that declares the profiled member.
      The first memeber that satisfies the constraints is used to obtain a pre-configured instance in order to invoked the profiled member.
    2. If no such attribute was found, it tries to find a parameterless instance constructor. This constrcutor can be langword_csharp_public, langword_csharp_private, or of any other access modifier.
    3. If no parameterles constructor was found and the profiled member is not a static member a MissingMemberException exception is thrown.
