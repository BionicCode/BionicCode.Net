    Class ProfilerBatchResult

    The result of a multi iteration benchmark session.
    The ProfilerBatchResult consolidates all ProfilerResult result items of a single benchmark run
    and provides meta data on hte collected data.

    ArgumentListCount

    The number of iterations the Profiler has run the specified operation.

    Declaration
    public int ArgumentListCount { get; }
    Property Value
    Type Description
    Int32

    The number of iterations (which is equivalent to the number of results) per argument list.
    Remarks

    Using the profiler with annotations (attributes) allows to specify argument lists to simulate real-world behavior where usually the values of the arguments that are provided to the member change.
    A member is executed and profiled with each argument list for the number of the IterationCount times.
    For example if a method is profiled with three argument lists and an iteration count of 10, the IterationCount will return 10 and the total iterations run for the profiled member is the product of IterationCount and ArgumentListCount.
