    EnumerateNormDist(Double, Double, Double)

    Calculates the normal distribution and retuns an enumerable collection. The interval used is [µ-5σ...µ+5σ], where µ=mean and σ=standard deviation.

    Declaration
    public static IEnumerable<CartesianPoint> EnumerateNormDist(double mean, double standardDeviation, double resolution = 0.1)
    Parameters
    Type Name Description
    Double mean

    The arithmetic mean of the data series.
    Double standardDeviation

    The sigma i.e. standard deviation of the data series.
    Double resolution

    The factor to calculate the increments of the 'x' values used to calculate the normal distribution.
    The smallest absolute value of the range is multiplied with the resolution value.
    Returns
    Type Description
    IEnumerable<CartesianPoint>

    The data set of x,y points. Can be used to plot the graph.
    The data set always contains the mean µ (0σ), ±1σ, ±2σ, ±3σ.
    Remarks

    EnumerateNormDist(Double, Double, Double) and NormDist(Double, Double, Double) differ as follows: When you use EnumerateNormDist(Double, Double, Double), you can start enumerating the collection of cartesian points before the whole collection is returned.
    When you use NormDist(Double, Double, Double), you must wait for the whole array of cartesian points to be returned before you can access the array.
    Therefore, when you are expecting a huge spread of values i.e. a big standard deviation value for standardDeviation, EnumerateNormDist(Double, Double, Double) can be more efficient.
