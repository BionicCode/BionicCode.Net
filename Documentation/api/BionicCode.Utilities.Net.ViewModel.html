    CreateProgressReporterFromUiThread()

    Creates a System.IProgress<T> instance that is associated with the caller's thread. The registered progress callback is the virtual BionicCode.Utilities.Net.Common.ViewModelCommon.OnProgress(BionicCode.Utilities.Net.ProgressData) member.

    public IProgress<ProgressData> CreateProgressReporterFromUiThread()
    Returns
    Type Description
    System.IProgress<BionicCode.Utilities.Net.ProgressData>

    A System.IProgress<T> instance that always posts progress to the UI thread.
    Remarks

    The returned System.IProgress<T> instance is associated with the application's primary dispatcher thread. Progress is always reported to the UI thread that is associated with the Dispatcher returned by Application.Current.Dispatcher.
