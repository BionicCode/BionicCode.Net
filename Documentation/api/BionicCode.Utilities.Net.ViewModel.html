    CreateProgressReporterFromUiThread()

    Creates a IProgress<T> instance that is associated with the caller's thread. The registered progress callback is the virtual OnProgress(ProgressData) member.

    Declaration
    public IProgress<ProgressData> CreateProgressReporterFromUiThread()
    Returns
    Type Description
    IProgress<ProgressData>

    A IProgress<T> instance that always posts progress to the UI thread.
    Remarks

    The returned IProgress<T> instance is associated with the application's primary dispatcher thread. Progress is always reported to the UI thread that is associated with the Dispatcher returned by Application.Current.Dispatcher.
