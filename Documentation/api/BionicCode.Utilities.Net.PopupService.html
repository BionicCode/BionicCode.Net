    IsStickyProperty

    When set to true, the System.Windows.Controls.Primitives.Popup is forced to stick to the current System.Windows.Controls.Primitives.Popup.PlacementTarget. The System.Windows.Controls.Primitives.Popup will follow the System.Windows.Controls.Primitives.Popup.PlacementTarget whenever it changes it's screen coordinates.
